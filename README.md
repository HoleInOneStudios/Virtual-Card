# Virtual Card

## Url parameters

| Parameter  | Description                         |
| ---------- | ----------------------------------- |
| `to`       | who the card is to                  |
| `from`     | who the card is from                |
| `occasion` | what is the occasion                |
| 'color`    | what should the background color be |

## Example

 > <http://www.benbowles.me/Virtual-Card?occasion=Mother%27s+Day&to=Mom&from=You+Children&color=skyblue>
