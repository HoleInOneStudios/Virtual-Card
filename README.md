# Virtual Card

## Url parameters

| Parameter  | Description                                  |
| ---------- | -------------------------------------------- |
| `to`       | who the card is to                           |
| `from`     | who the card is from                         |
| `occasion` | what is the occasion                         |
| `color`    | what should the background color be          |
| `note`     | special note to be displayed under the title |
